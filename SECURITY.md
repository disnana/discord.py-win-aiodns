# Security Policy / Security Contact

## Supported versions

Security fixes are applied to the latest released version.

## Reporting a vulnerability

Please do not publish a proof of concept or a bot token in a public issue.
Open a private GitHub security advisory for this repository, or contact the
maintainer through the contact method listed on the repository profile.

Please include the affected version, the impact, and clear reproduction steps.

---

## セキュリティ報告

セキュリティ上の問題、再現コード、bot token を公開 Issue に投稿しないでください。
このリポジトリの private security advisory を使うか、リポジトリプロフィールに記載された連絡先から報告してください。

対象バージョン、影響、再現手順を添えてください。
