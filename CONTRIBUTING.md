# Contributing / コントリビューション

Thank you for considering a contribution.

- Keep changes focused and include tests for changed behaviour.
- Run `python -m unittest discover -s tests -v` before opening a pull request.
- Do not include bot tokens, personal DNS configuration, or generated build
  artifacts in commits.
- Describe Windows version, Python version, aiodns version, and pycares version
  when reporting resolver behaviour.

## 日本語

コントリビューションを歓迎します。

- 変更は一つの目的に絞り、挙動を変える場合はテストを追加してください。
- Pull Request 前に `python -m unittest discover -s tests -v` を実行してください。
- bot token、個人の DNS 設定、生成済み配布物を commit に含めないでください。
- resolver の問題は Windows、Python、aiodns、pycares の各バージョンを添えて報告してください。
